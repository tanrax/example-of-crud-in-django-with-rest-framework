## Install

``` bash
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
python3 manage.py migrate
```

## Run

``` bash
python3 manage.py runserver
```

## Fake data

``` bash
python3 manage.py runscript create_books
```

## Test

``` bash
pytest
```
